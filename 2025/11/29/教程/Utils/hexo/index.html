编写格式

type 属性

在题头部分，type 属性有特殊作用，一般会被识别成特定的页面，比如使用 hexo new page tags 的时候，就会在 source 下生成一个 tags 目录和一个 index.md 文件，此时只需要在 index.md 页面的题头部分编写 type: tags 的话，那么该页面就会对所有文章的标签进行汇总和整理，对应 menu 的 /tags 页面。

Hexo 插件

hexo-image-link

存在图片 404 的问题？可以按照如下步骤解决：

  1. 在 _config.yaml 中，将 post_asset_folder 设置为 true
  2. 安装 npm install hexo-image-link --save 依赖
  3. 确保图片所在的位置在与文档同级的同名目录下：./${filename}/xxx.png
